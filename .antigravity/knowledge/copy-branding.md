# Copy & Branding — Autarch

## Identity
- **Name:** Autarch
- **Tagline:** The Ultimate Agentic Shell
- **Color Palette Requirements:** Deep blacks (#000000), sharp contrast (white), subtle brand accent (usually terminal-tinted green or corporate grey). Do not use soft pastels.

## Vocabulary
- **Workspace** (not folder)
- **Agentic Shell** (not command line or IDE wrapper)
- **Persona** (not agent prompt)
- **Orchestration** (not managing)

We prioritize a feeling of overwhelming capability and absolute control. The UI should make the user feel like a commander-in-chief in a futuristic command center.
