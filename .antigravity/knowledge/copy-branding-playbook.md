## Antigravity Brand Identity

### Marken-Kern
Antigravity ist eine Premium SaaS-/AI-Application.
Kein generisches Tool. Ein intelligentes System für Nutzer, die Wert auf Performance, Sicherheit und Design legen.

### Tone of Voice: Calm Confidence

| Eigenschaft | Beschreibung | Beispiel |
|---|---|---|
| **Calm** | Keine Panik, keine Dringlichkeit | ✅ "Dein System zeigt Dissonanz" ❌ "ACHTUNG! Deine Werte sind schlecht!" |
| **Confident** | Weiß was es tut, keine Hedge-Wörter | ✅ "Die Simulation zeigt..." ❌ "Vielleicht könnte es sein, dass..." |
| **Premium** | Exklusiv, nicht laut | ✅ "Dein Digital Twin hat neue Signale" ❌ "Hey! Neue Daten verfügbar!" |
| **Technical** | Präzise aber verständlich | ✅ "Spezifische Metrik bei 73%" ❌ "Wert okay" |

### Calm Confidence Scale (1-5)

```
1 — Zu passiv:     "Es könnten vielleicht neue Daten da sein..."
2 — Unsicher:      "Wir glauben, dass deine Werte sich verbessert haben."
3 — ✅ PERFEKT:    "Dein Digital Twin zeigt eine aufsteigende Trajectory."
4 — Zu direkt:     "Du hast 12 neue Follower. Tu mehr davon."
5 — Zu aggressiv:  "DEINE WERTE EXPLODIEREN! 🚀🔥"
```

## Verbotene Begriffe (Hard Rules)

| ❌ Verboten | Begründung | ✅ Alternative |
| `Dashboard` | Generisch | `Cockpit` |
| `Datenpunkte` | Kalt | `Signale` |
| `Score` (roh) | Ohne Kontext | `Trajectory` |
| `Klicken` | Oldschool | `Starten`, `Erkunden`, `Aktivieren` |

## Antigravity-Lexikon (Pflicht-Metaphern)

### Core-Metaphern

| Begriff | Bedeutung | UI-Kontext |
|---|---|---|
| **Cockpit** | Haupt-Dashboard | Navigation |
| **Trajectory** | Verlaufskurve / Entwicklungspfad | Metriken-Anzeigen |

|---|---|

## CTA-Label-Katalog

### Do's
```
✅ "Simulation starten"
✅ "Twin kalibrieren"
✅ "Course Correction ansehen"
✅ "Signale auswerten"
✅ "Cockpit öffnen"
✅ "Trajectory verfolgen"
```

### Don'ts
```
❌ "Hier klicken"
❌ "Mehr erfahren" (zu generisch)
❌ "Vorgang starten"
❌ "Datenbankabfrage"
❌ "Jetzt kaufen!" (zu aggressiv)
```
