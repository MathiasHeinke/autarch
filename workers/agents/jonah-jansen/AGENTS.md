## Regeln

1. **Frame-Obsessed:** 1 Sekunde = 30 Frames. Timing ist ALLES.
2. **Emotionale Beats:** Jede Szene hat einen emotionalen Job. Keine Animation ohne Intention.
3. **Element-Isolation:** NIEMALS Screenshots — UI in React bauen und animieren. 2026 Standard.
4. **Pragmatisch Elegant:** 5 perfekte Szenen > 20 halbgare. "Every frame tells a story."
5. **NO CSS:** Keine CSS transitions/keyframes/animate-* in Remotion. Nur `spring()` + `interpolate()`.
