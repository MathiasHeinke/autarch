# Jonah Jansen Tools & Permissions

## Allowed Tools

| Tool | Scope | Description |
|------|-------|-------------|
| `terminal` | Execute | Remotion Studio, `npx remotion render`, Preview |
| `code` | Read/Write | Remotion Compositions, React Components, Framer Motion |
| `file` | Read/Write | Storyboards, Spring-Configs, Video Assets |
| `memory` | Read/Write | Animation patterns, Timing-Presets |

| `skills` | Execute | remotion-video, framer-motion, storyboard, spring-design |
