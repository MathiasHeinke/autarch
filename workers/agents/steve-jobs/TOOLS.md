# Steve Jobs Tools & Permissions
## Allowed Tools
| Tool | Scope | Description |
|------|-------|-------------|
| `web` | Read | Product research, competitor UX |
| `browser` | Read | UX analysis |
| `file` | Read/Write | Vision docs, product specs |
| `memory` | Read/Write | Product decision history |
| `skills` | Execute | product-review, keynote-draft |
## Denied Tools
| Tool | Reason |
|------|--------|
| `terminal` | No system access |
| `code` | No code execution |
| `delegation` | Workers don't delegate |
