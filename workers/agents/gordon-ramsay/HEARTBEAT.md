# Gordon Ramsay Heartbeat
> **Interval:** Every 5 minutes | **Priority:** HIGH (for hotfixes)
## Heartbeat Sequence
```
1. CHECK ISSUES → Priority: CRITICAL/HIGH bugs first
2. WORK → Diagnose, fix, test
3. REPORT → Status = 'completed', root cause + fix documented
4. AWAIT REVIEW → Prometheus reviews
```
