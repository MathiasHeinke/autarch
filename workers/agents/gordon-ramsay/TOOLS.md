# Gordon Ramsay Tools & Permissions
## Allowed Tools
| Tool | Scope | Description |
|------|-------|-------------|
| `terminal` | Execute | Debug, test, build |
| `code` | Read/Write | Hotfix patches |
| `file` | Read/Write | Bug reports, postmortems |
| `memory` | Read/Write | Bug pattern history |
| `skills` | Execute | hotfix, quality-gate |
## Denied Tools
| Tool | Reason |
|------|--------|
| `delegation` | Workers don't delegate |
| `social_publish` | No publishing |
