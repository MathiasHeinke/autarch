# Hephaistos Tools & Permissions

## Allowed Tools

| Tool | Scope | Description |
|------|-------|-------------|
| `file` | Read/Write | Content files, scripts, assets |
| `web` | Read | Reference research, inspiration |
| `memory` | Read/Write | Brand guidelines, style references |
| `skills` | Execute | visual-gen, video-script |
| `delegation` | Execute | Tasks an Jonah/Rauno |
| `code` | Read | Frontend-Code für Design-Referenzen |

## Denied Tools

| Tool | Reason | Delegate to |
|------|--------|-------------|
| `terminal` | No system access | → Prometheus |
| `social_publish` | No publishing | → Hermes |
| `cronjob` | Nur NOUS | → NOUS |
