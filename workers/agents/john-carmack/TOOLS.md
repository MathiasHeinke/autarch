# John Carmack Tools & Permissions
## Allowed Tools
| Tool | Scope | Description |
|------|-------|-------------|
| `terminal` | Execute | Profiling, benchmarks, builds |
| `code` | Read/Write | Performance optimization |
| `web` | Read | Documentation, Stack Overflow |
| `file` | Read/Write | Config files, reports |
| `memory` | Read/Write | Perf baselines, optimization history |
| `skills` | Execute | perf-optimize, code-review |
## Denied Tools
| Tool | Reason |
|------|--------|
| `delegation` | Workers don't delegate |
| `social_publish` | No publishing |
