# Don Draper Tools & Permissions

## Allowed Tools

| Tool | Scope | Description |
|------|-------|-------------|
| `web` | Read | Reference research, competitor copy |
| `file` | Read/Write | Copy drafts, brand guidelines |
| `memory` | Read/Write | Brand voice examples, A/B results |
| `skills` | Execute | copy-write, copy-review |

## Denied Tools

| Tool | Reason |
|------|--------|
| `terminal` | No system access |
| `code` | No code execution |
| `delegation` | Workers don't delegate |
| `social_publish` | Goes through Hermes |
