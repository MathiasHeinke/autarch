# 🥃 Draper Copy — Copy Review Protocol

## When to Use
- Writing UI text, labels, CTAs
- Marketing material, landing pages
- Onboarding flows, error messages
- Any user-facing text that needs to convert

## Procedure

### 1. Activate Draper
`/personality draper`

### 2. Brief
- What screen/context?
- What should the user FEEL?
- What should the user DO?
- What's the brand voice? (if established)

### 3. Insight
Identify the psychological lever:
- Control? ("You decide")
- Exclusivity? ("Not for everyone")
- Progress? ("You're ahead")
- Safety? ("Protected, always")
- Status? ("The smart choice")

### 4. Generate 3 Variants
| Variant | Style | Copy |
|---------|-------|------|
| A: Safe | On-brand, minimal risk | ... |
| B: Bold | Pushing boundaries | ... |
| C: Wild | Surprising, potentially polarizing | ... |

### 5. Headline Test
Can each variant be reduced to ≤7 words?
If not, simplify until it can.

### 6. Compliance Check (if applicable)
- No prohibited terms (diagnose, guarantee, cure, prove)
- No aggressive CTAs ("BUY NOW!", "LAST CHANCE!")
- Tone matches brand guidelines

## Pitfalls
- Don't write copy without context (which screen? which user moment?)
- Don't default to generic labels ("Click here", "Learn more", "Submit")
- Every label is an opportunity for emotional connection
